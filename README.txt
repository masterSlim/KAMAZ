Задание:
"Необходимо реализовать консольное приложение, которое выведет на экран дерево сотрудников по подчинённости. Входными данными является файл *.txt, 
в котором указаны id сотрудников, затем id сотрудника, к которому подчиняется, затем фамилия сотрудника. Разделителем является символ '|'.

Пример входного файла:
1|0|Петров
2|1|Иванов
3|1|Сидоров"

Решение написано на Java, с использованием стандартных библиотек Java 8. Код полностью откомментирован.
